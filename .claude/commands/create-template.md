# 创建新模板

帮助用户快速创建 MD2Anything 的新模板。

## 使用方式

```
/create-template
```

## 执行步骤

当用户调用此技能时，请按以下步骤操作：

### 1. 收集模板信息

通过交互式问答收集以下信息：

**必填信息：**
- 模板类型（wechat / xiaohongshu / email / resume / general）
- 模板名称（显示名称，如"科技蓝"）
- 模板 ID（唯一标识，如"wechat-tech-blue"）
- 描述（简短描述模板特点）

**样式偏好（可选，有默认值）：**
- 主色调（默认根据类型推荐）
- 背景色（默认 #ffffff）
- 字体风格（现代 / 经典 / 简约）

### 2. 生成模板代码

根据收集的信息，生成完整的模板对象代码：

```typescript
{
  id: '{{模板ID}}',
  name: '{{模板名称}}',
  format: '{{模板类型}}',
  description: '{{描述}}',
  themeColors: ['{{主色调}}', '{{辅助色}}', '{{背景色}}'],
  styles: {
    // 根据类型和偏好生成的样式
  },
}
```

### 3. 写入文件

将模板添加到对应的模板文件中：
- 微信 → `src/templates/wechat.ts`
- 小红书 → `src/templates/xiaohongshu.ts`
- 邮件 → `src/templates/email.ts`
- 简历 → `src/templates/resume.ts`
- 通用 → `src/templates/general.ts`

### 4. 提供测试指引

创建完成后，提供以下测试步骤：
1. 启动开发服务器 `npm run dev`
2. 选择对应格式
3. 选择新创建的模板
4. 测试导出功能

## 模板类型说明

### wechat（微信）
- 必须使用内联样式
- 适合手机阅读，字体适中
- 支持代码高亮

### xiaohongshu（小红书）
- 字体要大（视觉冲击力）
- 边距充足
- 颜色鲜明

### email（邮件）
- 使用 table 布局
- 完全内联样式
- 兼容邮件客户端

### resume（简历）
- 专业正式
- 结构清晰
- 适合 PDF 导出

### general（通用）
- 通用场景
- 支持多种导出格式
- 样式灵活

## 样式生成规则

根据模板类型自动生成合适的样式：

1. **container** - 根据类型设置合适的 padding 和 line-height
2. **标题样式** - h1/h2/h3 使用主色调
3. **引用样式** - 左边框使用主色调
4. **代码样式** - 代码块使用深色背景
5. **表格样式** - 使用主色调作为表头背景
6. **链接样式** - 使用主色调
